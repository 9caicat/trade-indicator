# trade-indicator
韭菜老猫自研交易指标
钻石底指数 — 用于评估A股宽基指数的市场情绪
基于 200 日与 850 日双均线偏离度的底部识别模型

**核心公式**
钻石底指数 = (现价 / MA200) × √(现价 / MA850)

| 参数              | 含义                                     |
| --------------- | -------------------------------------- |
| **MA200**       | 200日移动平均线，代表近7个月投资者平均持仓成本（短期锚点）        |
| **MA850**       | 850日移动平均线（约3.5年），覆盖完整库存周期与宏观政策周期（长期锚点） |
| **√(现价/MA850)** | 对长期因子开平方，降低850日均线波动过慢带来的钝化效应           |

钻石底阈值：各指数历史数据的 5% 分位数 —— 仅 5% 的时间处于该极端低估区间。

⚠️ ##风险提示##
V型闪电底盲区：极速暴跌反弹时（如2020年3月），200日均线尚未充分下行，指标可能来不及触发信号 —— 需结合成交量、恐慌指数辅助判断
阈值漂移风险：若A股发生结构性变化（注册制深化、成分股大规模调整、ROE趋势性下移），历史分位数阈值可能失效，需定期重标定
个股不适用：该指标仅适用于宽基指数和行业ETF。单只股票可能因基本面恶化而永久跌破长期均线，不存在均值回归保证
**磨底时间成本：即使进入钻石底区域，市场也可能横盘数月（如2018年10月后磨底2个月）—— 建议以定投方式参与，避免一次性押注最低点
